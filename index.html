A quick, lightweight and easy-to-use i18n engine for static websites

<script src="https://cdn.jsdelivr.net/npm/vavilon@0.3.0/dist/vavilon.min.js"></script>

With vavilon.js translating static websites is just a matter of a few extra lines of code.

index.html

<html lang="en">
<head>
  <title>Hello World</title>
  <script src="ru.json" type="application/json" data-vavilon-dict="ru"></script>
  <script src="https://cdn.jsdelivr.net/npm/vavilon@0.3.0/dist/vavilon.min.js"></script>
</head>
<body>
  <h1 class="vavilon" data-vavilon="hello">Hello World!</h1>
  <hr>
  <button onclick="changeLocale('ru')">Russian</button>
  <button onclick="changeLocale('en')">English</button>
</body>
</html>

ru.json

{
    "hello": "Привет, мир!"
}
